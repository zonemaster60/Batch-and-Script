Addons
======

1. In order to use the 'ADDONS menu' you must first create a blank 'addons.txt'
2. Place the 'addons'txt' file in the 'files' folder.
3. You can edit this file and add '.exe' files that you access frequently.
4. Put your '.exe' files in the same folder as 'addons.txt' the 'files' folder.
(You can have up to 5 addons.)
